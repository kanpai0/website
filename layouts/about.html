乾杯 · Des mocktails pour tous les moments

Le nom

Kanpai (乾杯) signifie « santé » en japonais — un mot universel, joyeux, que tout le monde comprend. Ø, emprunté à l'esthétique scandinave, symbolise le zéro : pas d'alcool. Ensemble, ils forment quelque chose de simple et reconnaissable, unique sur le web.

Deux usages, une app

Pour l'invité

Se laisser guider par les goûts

Les pilules de saveurs filtrent les recettes. On parcourt comme un menu de bar — fruité, épicé, herbacé — sans savoir ce qu'on cherche.

Pour l'hôte

Partir de son frigo

Le panneau Mon frigo laisse décocher les ingrédients absents. On voit instantanément les recettes réalisables avec ce qu'on a.

La démarche

  • Deploy first Infrastructure validée en production avant la première feature.
  • Décisions documentées 20+ ADRs pour un projet solo — chaque choix de stack tracé et argumenté.
  • Qualité intégrée Pre-commit hook, tests de régression visuelle (Playwright), scénarios BDD (Gherkin).
  • Sobriété technique Hugo seul, zéro framework CSS, zéro npm en production.
  • CSS comme machine d'état Le panneau frigo fonctionne via :has(), sans JavaScript.
