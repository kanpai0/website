# Kanpai Ø — Résumé du projet

## Le concept

Site de **mocktails** (cocktails sans alcool) destiné aux adeptes du goût et des saveurs, avec une ambiance fun, healthy et conviviale. Projet en phase exploratoire — terrain de jeu, pas encore de produit lancé.

## Choix du nom

Après un brainstorming explorant le japonais, le maori, le sanskrit, le scandinave et le français créatif, le choix s'est porté sur **Kanpai Ø** :

- **Kanpai** (乾杯) = "santé !" / "tchin-tchin" en japonais, mot universellement reconnu
- **Ø** = zéro barré, symbolisant l'absence d'alcool, avec un look design scandinave
- Combinaison `kanpai0` vierge sur le web — aucune marque ni site existant
- Autres finalistes écartés : Skål Ø (trop de concurrents sur "Skål" dans l'univers boisson), kanpa.io (perd le sens du mot, connotation trop tech)

## Protection de marque

Pas de dépôt de marque à ce stade — disproportionné pour un projet exploratoire. Stratégie recommandée :

1. **Maintenant** : domaines + réseaux sociaux (fait)
2. **Si le projet se concrétise** : dépôt INPI (~190€, protection France, 10 ans)
3. **Si décollage international** : dépôt via le système de Madrid

Note : "kanpai" étant un mot courant en japonais, le "0" ajouté renforce la distinctivité pour un éventuel dépôt.
